### CSS3新特性

CSS3 是层叠样式表（Cascading Style Sheets）语言的最新版本，旨在扩展CSS2.1。

它带来了许多期待已久的新特性， 例如圆角、阴影、gradients(渐变) 、transitions(过渡) 与 animations(动画) 。以及新的布局方式，如 multi-columns 、 flexible box 与 grid layouts

1. 布局上：
- 媒介查询
- 响应式布局（flex、grid布局）
- 图片元素srcset属性
- css的object-fit属性


2. 视觉上
- 圆角、阴影、渐变
- transform变换
- filter滤镜
- animation
